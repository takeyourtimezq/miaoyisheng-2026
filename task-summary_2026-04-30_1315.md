# Task Summary - 2026-04-30 13:15

## Objective
为铖烈的品牌营销公司6大服务类别（战略咨询、全案代运营、招商加盟、社群运营、培训赋能、品牌传播）设计"反常识框架"，用于客户提案展示。

## Key Reasoning
- 传统咨询框架（SWOT/4P等）同质化严重，无法在客户面前建立差异化认知
- 核心策略：每个框架都围绕"打破常识"设计——客户以为的问题不是真问题，真问题在常识的反面
- 六个框架统一底层心法：方向对但逻辑反了

## Conclusions
设计了6个反常识框架：
1. 战略咨询 → 逆向定位法（定位不是找自己是谁，是决定自己不是谁）
2. 全案代运营 → 不对称作战模型（80%GMV来自20%动作，其余是成本）
3. 招商加盟 → 反漏斗模型（先筛掉不合适的，比吸引更多更重要）
4. 社群运营 → 戒断测试法（活跃度是虚荣指标，戒断反应才是真实粘性）
5. 培训赋能 → 行为固化曲线（培训价值不在课堂，在课后21天）
6. 品牌传播 → 认知落差引擎（最好传播不是铺面，是制造信息差）

统一心法：客户以为的问题不是真问题，真问题藏在常识的反面。
