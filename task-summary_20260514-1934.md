# 轻刻时代项目推荐表时间可视化 - 工具推荐

## Objective
为轻刻时代"营养小站万店连锁新实体扩张项目"的四阶段推进计划，推荐飞书最佳时间可视化工具。

## Key Input
- 合同文件：《26.5.12轻刻品牌高级顾问合作协议修改版(3).docx》
- 项目四阶段：0-1店样板验证(45天) → 1-10店连锁起盘(45天) → 10-50店批量复制(90天) → 50-100店势能放大(180天)
- 总服务周期12个月，费用188万元
- 7大模块：招商体系、培训体系、线上账号资质、单店模型、门店设计、产品设备、资源势能

## Conclusion
推荐**飞书多维表格（Bitable）**，理由：
1. 甘特图/看板/时间线视图一键切换，兼顾内部推进和对外展示
2. 支持自动化提醒、公式计算进度、仪表盘汇总
3. 相比飞书项目更轻量、对外更友好；相比飞书文档有原生时间线能力
4. 可创建关联表实现阶段→交付物下钻查看

## 搭建方案
- 主表字段：阶段、模块、交付物、负责人、开始/结束日期、状态、优先级
- 三个视图：甘特图（推进）、看板（状态）、时间线（展示）
- 进阶：自动化提醒、进度公式、仪表盘、颜色编码
